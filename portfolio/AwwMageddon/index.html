AwwMageddon

AwwMageddon is a city destruction game created for my Game Design 325 class.
---------------------------------------------------------------
This was my first time making a fully 3D game and it incorporated a lot of different elements to it.

Footage and Gameplay

  • AwwMageddon was developed making in Unreal Engine 5 using the built-in blueprint system and other frameworks

  • The "Kaijuwu" is a cat-like monster that you can only influence in your mission to destroy the city
  • The Kaijuwu runs off of an AI behavior tree I set up to roam/fall asleep randomly if it is not tempted your array of toys

  • Also made use of in this project is the Unreal engine path-finding system (for the Kaijuwu to roam) and the Chaos Physics program utilized for the building destruction

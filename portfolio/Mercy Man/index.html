Mercy Man

This is a personal project of mine to recreate and modernize the mechanics of Resident Evil Gaiden(hereby REG) into a framework that can maybe, eventually be built into a full game.

Footage and Gameplay

The current focus of the project is to recreate and improve upon the original combat mechanics of REG

Currently developed is a crude inventory system using JSON save data and a test enemy with a combat scene
